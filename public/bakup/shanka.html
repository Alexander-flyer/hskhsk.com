Shanka 闪卡 Flashcards Web App

Beta testing is going well! You can test the app here by pointing your phone/tablet/PC/Mac's web browser to this page:

 http://shanka.hskhsk.com/beta1/

It will work on pretty much anything with a modern HTML5 web browser (yes IE11, even you, just about...).

To stay up to date with and discuss new features, bug reports, etc., join this group: facebook.com/groups/shankaapp or this discussion thread on chinese-forums.

Features Already Implemented

  • Display of cursive (joined up) and calligraphic fonts.
  • Shows stroke order animations and historic forms (oracle script, seal script, etc.) for many common characters from word info page.
  • All flashcard data is kept offline with no server needed. Only stroke order and character etymology require you to be online.
  • Freehand (cursive, joined up) hanzi drawing with no stroke-order detection (a good and a bad thing).
  • Can undo and erase strokes while drawing.
  • Cards can be studied, and rated on a 1-5 scale with customisable labels.
  • Rate multiple parts (e.g. pinyin and definition when shown the hanzi) at once
  • You can see your whole study history, and your upcoming queue, at any time.
  • Imports hanzi, pinyin, and definitions flashcard files exported by Skritter, Pleco (text, not XML), and StickyStudy.
  • Exports Pleco format, and imports/exports a custom format that allows all data to be exchanged between instances of the app.
  • Flexible configuration (there will be no complaining about not being able to hide definitions etc.) 
  • Practice mode for difficult characters, which tries to be the electronic equivalent of writing Hanzi over and over again on a character grid. 
  • Knowledge-rate-based SRS-style algorithm that keeps track of how well you know cards, and algorithmically works old cards into the study sessions.
  • No restrictions on using the back button while studying. - go back and review your previously studied cards.
  • The categories (lists) which contain a given word are visible, and can be browsed from the character info screen.
  • Words that share characters are shown on the character info screen.
  • Algorithm parameters can be tweaked, and multiple sets of parameters can be present and selected between to reflect how you are studying at that moment (learning new characters, revising for a test, just feel like some random characters, etc.).
  • Open word in Pleco  from word info screen (iOS only)

Known Bugs (Beta v0.22)

  • Mobile-optimised layout, should be improved for desktop use.
  • Doesn't collect or display progress statistics yet.
  • Question choice should be based on the level of knowledge of parts of the chosen card.

Possible future features

  • Progress graphs.
  • Sentences, both re-arrange words and fill-in-the-blanks.
  • Multiple choice quiz-style questions.
  • Skritter, Pleco, and Anki-style SRS algorithm based on scheduled date, not knowledge rate.
  • Link to online dictionaries, and Android dictionary if an API exists like with Pleco on iOS.
  • Import/Export of Pleco XML flashcard format, allowing progress to be imported/exported.
  • Use Skritter API and a Skritter subscription to download flashcards and/or sync progress.
  • Stroke order display and/or detection.
  • Automatic character rating and final stroke detection.
  • Use online dictionary to search for new flashcards, and fill in definitions & pinyin when adding cards.
